# Documentación Frontend – InAdvance Casas

## 1. Información General del Proyecto

- **Nombre del proyecto:** Ultracasas
- **Tipo:** Frontend con Vue 3 y TypeScript
- **Herramienta de construcción:** Vite
- **Estilos:** TailwindCSS
- **Estado global:** Pinia
- **Routing:** Vue Router

---

## 2. Estructura del Proyecto

- **App.vue:** Componente raíz de la aplicación
- **main.ts:** Punto de entrada principal
- **assets:** Recursos estáticos como imágenes y estilos
- **components:** Componentes reutilizables
- **views:** Páginas principales de la aplicación
- **router:** Configuración del enrutamiento
- **stores:** Manejo del estado global mediante Pinia

---

## 3. Instalación y Ejecución

### Requisitos

- Node.js versión 20 o superior
- npm, pnpm o yarn

### Comandos principales

- Instalación de dependencias
- Ejecución en modo desarrollo
- Construcción del proyecto para producción
- Previsualización de la build

---

## 4. Scripts Disponibles

- Ejecución del entorno de desarrollo
- Generación de build
- Previsualización del proyecto construido
- Ejecución de pruebas unitarias
- Verificación de tipos
- Ejecución de linters
- Formateo del código

---

## 5. Rutas del Proyecto (Vue Router)

El sistema de rutas define las páginas disponibles dentro de la aplicación y se encuentra dentro del directorio encargado del enrutamiento.

---

## 6. Manejo del Estado Global (Pinia)

El estado global del proyecto se gestiona mediante stores organizados dentro del directorio correspondiente a Pinia.

---

## 7. Estilos del Proyecto (TailwindCSS)

TailwindCSS es el sistema de estilos utilizado para la aplicación

---

## 8. Herramientas de Calidad de Código

El proyecto incorpora herramientas para mantener un estilo de código consistente y detectar errores:

- ESLint
- Prettier
- Oxlint
- Verificación de tipos con TypeScript y vue-tsc

---

## 9. Tecnologías Principales Utilizadas

- Vue 3
- Vue Router
- Pinia
- TailwindCSS
- Vite
- TypeScript
- Vitest
